# Intelligent Email Security and Spam Filtering System

A Flask-based web application for detecting spam and phishing emails using Machine Learning (XGBoost + TF-IDF + URL Analysis).

## Features
- Email classification: Safe / Spam / Phishing
- URL reputation analysis
- Confidence score display
- Analysis history (SQLite)
- Evaluation metrics dashboard

## Tech Stack
Python, Flask, XGBoost, scikit-learn, SQLite, HTML/CSS

## Run
```bash
pip install -r requirements.txt
python app.py
```
